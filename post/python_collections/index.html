collections

数据类型—容器数据类型

这个模块实现了特定目标的容器，以提供Python标准内建容器 dict , list , set , 和 tuple 的替代选择。

Object description
namedtuple() 创建命名元组子类的工厂函数
ChainMap 类似字典(dict)的容器类，将多个映射集合到一个视图里面
deque 类似列表(list)的容器，实现了在两端快速添加(append)和弹出(pop)
Counter 字典的子类，提供了可哈希对象的计数功能
OrderedDict 字典的子类，保存了他们被添加的顺序
defaultdict 字典的子类，提供了一个工厂函数，为字典查询提供一个默认值
UserDict 封装了字典对象，简化了字典子类化
UserList 封装了列表对象，简化了列表子类化
UserString 封装了列表对象，简化了字符串子类化

  • collections.Counter

一种计数工具能够简单快速的计数。一个Counter还是一个dict的子类，用于计数可哈希对象，返回的是一个dict，元素像字典键(key)一样存储，它们的计数存储为value。

初始化：Counter(iterable-or-mapping)

1
2
3
4

>>> c = Counter()                           # a new, empty counter
>>> c = Counter('gallahad')                 # a new counter from an iterable
>>> c = Counter({'red': 4, 'blue': 2})      # a new counter from a mapping
>>> c = Counter(cats=4, dogs=8)             # a new counter from keyword args

字典接口，c[key] = value，若引用的key没有记录，则返回的0。

1
2
3
4

>>> c = Counter(['eggs', 'ham'])
>>> c['bacon']                              # count of a missing element is zero
0
c[key] += 1

most_common([n])，返回一个列表，其中包含 n 个最常见的元素及出现次数，按常见程度由高到低排序。 如果 n 被省略或为 None，most_common()将返回计数器中的 所有 元素。 计数值相等的元素按首次出现的顺序排序：

1
2

>>> Counter('abracadabra').most_common(3)
[('a', 5), ('b', 2), ('r', 2)]

subtract([iterable-or-mapping])

从 迭代对象 或 映射对象 减去元素。像dict.update()但是是减去，而不是替换。输入和输出都可以是0或者负数。

1
2
3
4
5

>>> c = Counter(a=4, b=2, c=0, d=-2)
>>> d = Counter(a=1, b=2, c=3, d=4)
>>> c.subtract(d)
>>> c
Counter({'a': 3, 'b': 0, 'c': -3, 'd': -6})

elements()

返回一个迭代器，其中每个元素将重复出现计数值所指定次。 元素会按首次出现的顺序返回。 如果一个元素的计数值小于一，elements()将会忽略它。

1
2
3

>>> c = Counter(a=4, b=2, c=0, d=-2)
>>> sorted(c.elements())
['a', 'a', 'a', 'a', 'b', 'b']

  • collections.OrderedDict

有序的字典和常规的词典一样，但额外有一些排序的功能。python 3.7以前，对dict做迭代时，我们无法确定Key的顺序，所以使用的OrderedDict。

一些不同：

  • 常规的dict擅长映射，而OrderedDict更擅长频繁的排序，其空间效率，迭代速度是次要的。

  • collections.nameturple()

nameturple()是一个函数，能够创建turple的子类，可以定义turple元素的个数，并且可以通过属性而不是索引来引用turple的元素。

nameturple(typename,field_names,*,rename=False,defaults=None)，举个栗子：

  • collections.deque()

    • append(x)

    添加 x 到右端。

    • appendleft(x)

    添加 x 到左端。

    • clear()

    移除所有元素，使其长度为0.

    • copy()

    创建一份浅拷贝。3.5 新版功能.

    • count(x)

    计算 deque 中元素等于 x 的个数。

    • pop()

      移去并且返回一个元素，deque 最右侧的那一个。 如果没有元素的话，就引发一个 IndexError。

    • popleft()

      移去并且返回一个元素，deque 最左侧的那一个。 如果没有元素的话，就引发 IndexError。

    • remove(value)

      移除找到的第一个 value。 如果没有的话就引发 ValueError。

1
2
3
4
5
6

>>> q = deque([1,2,3])
>>> q.append('x')
>>> q.append('x')
>>> q.appendleft('y')
>>> q
deque(['y', 1, 2, 3, 'x', 'x'])
