conda常用命令

首先理解下什么是conda，conda是一个开源包和环境的管理工具，可用于安装多个版本的软件包及其依赖关系，并在它们之间切换。使用conda可以解决python包依赖的问题以及python版本的问题，在安装各种包上非常方便，比如，conda install -c bioconda seqkit。

conda, Anaconda, Miniconda的关系：

  • conda包含在Anaconda和Miniconda中
  • Anaconda是python的发行版，内置众多python的包和附加的软件。包括conda, conda-build, python和超过150自动安装的科学包及其依赖项
  • Miniconda是一个小的引导版本，包含conda, python和它的依赖包

2. 环境管理

  • 查看环境管理的全部命令帮助
1

conda env -h
  • 查看当前系统的全部环境
1
2

conda env list 
conda info -e
  • 创建一个新的环境 至少指定一个python版本或安装的包
1
2

conda create -n env_name python=3.8 package_1 package_2 
# -n = --name
  • 激活某个环境
1

source activate env_name
  • 退出某个环境
1

conda deactivate
  • 复制某个环境
1

conda create new_env_name old_env_name
  • 删除某个环境
1
2

conda remove env_name
conda remove -n env_name --all

#添加数据源：例如, 一些包下载过慢，可以添加清华anaconda镜像：

#删除数据源
