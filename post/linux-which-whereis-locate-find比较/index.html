which

  • which，通过PATH环境变量找到路径内查找可执行文件，基本功能是查找可执行文件
1
2

which passwd
#/usr/bin/passwd

whereis

  • whereis，查看文件位置，因为linux系统会将系统内所有的文件都记录在一个数据库文件中当时有whereis和locate，会从数据库中查找数据。但数据库文件不是实时更新的，默认可能一周更新一次。
    • -b，查二进制文件
    • -m，说明文档
    • -s，找源文件
1
2

whereis passwd
#passwd: /usr/bin/passwd /etc/passwd

locate

  • locate，文件或目录

find

  • find，通过遍历硬盘；最强大，什么都能查，根据条件查找文件，在硬盘上查找，效率很低
1

find path options

时间查找参数： -atime n :将n24小时内存取过的的文件列出来 -ctime n :将n24小时内改变、新增的文件或者目录列出来 -mtime n :将n*24小时内修改过的文件或者目录列出来 -newer file ：把比file还要新的文件列出来 名称查找参数： -gid n ：寻找群组ID为n的文件 -group name ：寻找群组名称为name的文件 -uid n ：寻找拥有者ID为n的文件 -user name ：寻找用户者名称为name的文件 -name file ：寻找文件名为file的文件（可以使用通配符）
