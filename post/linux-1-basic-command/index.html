Basic commands

学习参考

命令 描述
ls List 显示文件列表
cd change directory 切换目录
mkdir make directory 创建一个空文件夹
less 翻页查看文件
head 默认查看文件前10行
tail 默认查看文件后10行
cat 直接查看整个文件
cut 取出文件特定的列或字符
sort 排序
uniq 去重复
chmod 改变权限
mv Move 移动文件或目录，重命名文件或目录
rm remove 删除目录或文件
cp copy and paste 复制文件或目录
touch 创建文件
pwd print working directory 显示当前工作目录
> 将结果输入给新文件或覆盖源文件
» 将终端结果输出给文件，内容会加在原文件尾部
| 将左边命令的标准输出（standard output）作为右边命令接受的标准输入（standard input）
